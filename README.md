TrekFinder is a backend service for a trekking discovery platform that helps trekkers easily find upcoming treks without searching across multiple social media platforms.

### Features
- Create and list trekking events
- Search and filter treks by month and difficulty
- WhatsApp-based contact with trek leaders
- Clean REST API architecture

### Tech Stack
- Node.js
- Express.js
- MongoDB
